# 🧠 AI 7일차: 이미지 파운데이션 모델

---

## 📘 1. 이미지 파운데이션 모델

### 1) AI 파운데이션 모델 개념 및 대표 모델

#### 🔹 파운데이션 모델이란?
- **파운데이션 모델**은 다양한 작업을 위한 **기본이 되는 모델**을 의미합니다.  
- 주로 **멀티모달** 작업에서 **전이 학습(transfer learning)**을 통해 다양한 응용을 지원하며, 여러 도메인에서 활용됩니다.

#### 🔹 CLIP 구조, 학습, 응용
- **CLIP**(Contrastive Language-Image Pretraining)은 **이미지와 텍스트**를 연결하여, 이미지와 텍스트 간의 **공통된 의미**를 학습합니다.
- **구조**: 텍스트와 이미지 각각에 대해 **임베딩 공간**을 학습하여 두 가지 데이터 유형을 **대조적 방식**으로 연결합니다.
- **학습**: 이미지와 텍스트의 쌍을 통해 두 공간을 연결하는 **대비 학습** 방식으로 진행됩니다.
- **응용**: 이미지 검색, 이미지 캡션 생성, 멀티모달 분석 등 다양한 작업에 활용됩니다.

---

### 2) Vision-Language Model (VLM)

#### 🔹 SigLIP - 멀티모달 정합 응용
- **SigLIP**(Signal Language-Image Pretraining)은 **이미지와 텍스트** 간의 상호작용을 통해 더욱 **정확한 정보**를 추출하는 모델입니다.
- 주로 **멀티모달 정합** 작업에 사용되며, 텍스트와 이미지 간의 **연관성**을 학습하여 두 데이터 간의 **정확한 매칭**을 가능하게 합니다.

#### 🔹 멀티 모달 언어 모델
- **멀티모달 언어 모델**은 **텍스트**, **이미지**, **음성** 등 다양한 유형의 데이터를 결합하여 **종합적인 정보**를 처리합니다.  
- 이러한 모델은 여러 형식의 입력을 동시에 처리하여 **상호보완적인 결과**를 도출할 수 있습니다.

#### 🔹 LLaVA
- **LLaVA**(Language and Vision Alignment) 모델은 **언어와 이미지 간의 관계**를 깊이 있게 학습하여, 두 가지 유형의 데이터를 **효율적으로 통합**합니다.
- **응용**: 텍스트와 이미지를 동시에 이해해야 하는 **이미지 캡셔닝**, **질문 응답 시스템** 등에 사용됩니다.

#### 🔹 최신 공개 VLM 모델 (Qwen)
- **Qwen**은 **Vision-Language Model** 중 최신 공개 모델로, **이미지**와 **언어**를 동시에 이해하고 처리할 수 있는 **강력한 성능**을 자랑합니다.
- 다양한 **도메인**에서 성능을 평가할 수 있으며, **멀티모달 학습**의 최전선에서 사용되고 있습니다.

#### 🔹 VLM의 성능을 높이는 트릭
- **학습 전략**: VLM의 성능을 향상시키기 위해 **대규모 데이터셋**과 **정교한 학습 기법**을 활용합니다.
- **트릭**:
  - **하이퍼파라미터 튜닝**: 모델의 성능을 최적화하기 위한 다양한 실험이 필요합니다.
  - **전이 학습**: 다른 모델에서 학습된 파라미터를 **전이하여** 빠른 수렴과 성능 향상을 유도합니다.

#### 🔹 도메인 특화 파운데이션 모델들
- **의료**: 의료 이미지 분석, 질병 예측 등에서 **의료 특화 모델**이 활용됩니다.
- **제조업**: 제품 이미지, 제조 공정에 대한 **정확한 예측**을 위해 사용됩니다.
- **3D 언어 모델**: 3D 데이터와 언어를 결합하여 **시뮬레이션**과 **가상 환경**에서의 예측을 가능하게 합니다.
- **로봇 행동 모델**: 로봇의 **동작**과 **언어**를 결합한 모델로, **자율 주행** 및 **로봇 학습**에 사용됩니다.

---

### 3) Small VLM과 파운데이션 모델들 소개

#### 🔹 sVLM (Small Vision-Language Models)
- **sVLM**은 **경량화된 VLM**으로, **모바일 장치**나 **제약된 환경**에서 사용될 수 있도록 최적화된 모델입니다.
- 더 적은 **연산 자원**을 사용하면서도 **고성능**을 유지할 수 있습니다.

#### 🔹 한국어 sVLM
- 한국어 데이터를 기반으로 학습된 **sVLM** 모델은 **다양한 한국어 데이터**에 특화되어, **한국어 텍스트**와 **이미지** 간의 관계를 잘 처리합니다.

---

### 4) 개인화, 합성 데이터 활용 사례

#### 🔹 파운데이션 모델의 주 응용 방법 (적응 학습)
- **적응 학습(Adaptation Learning)**: 파운데이션 모델은 다양한 **도메인**에 맞게 **적응 학습**을 통해, 특정 도메인에서 더 나은 성능을 발휘할 수 있습니다.
- 이 방식은 기존 모델을 **도메인 특화** 데이터로 재학습시켜 **성능을 개선**하는 데 사용됩니다.

#### 🔹 데이터 효율화를 위한 합성 데이터 사용
- **합성 데이터**는 실제 데이터를 기반으로 **인위적으로 생성**된 데이터입니다.  
- 모델 학습에 **데이터가 부족**할 때, 합성 데이터를 사용하여 **데이터 효율성**을 높이고, **모델 성능**을 향상시킬 수 있습니다.

#### 🔹 What's next?
- **미래**에는 더 **강력한 파운데이션 모델**들이 등장하여, **모든 산업**에서 **멀티모달** 데이터를 처리할 수 있게 될 것입니다.  
- 또한, **지속적인 데이터 효율화**와 **맞춤형 학습 전략**이 중요해질 것입니다.

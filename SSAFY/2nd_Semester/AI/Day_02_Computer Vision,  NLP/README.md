## 컴퓨터 비전이란? (Computer Vision)

### 📌 컴퓨터 비전 개요
컴퓨터 비전은 **이미지나 영상과 같은 시각적 데이터를 컴퓨터가 이해하고 처리하도록 하는 기술 분야**입니다.  
사람이 눈으로 인식하는 정보를 디지털 데이터 형태로 분석하여,  
의미 있는 정보로 변환하는 것을 목표로 합니다.

컴퓨터 비전은 인공지능(AI)의 한 분야로,  
머신러닝 및 딥러닝 기법과 결합되어 발전해 왔습니다.

---

### 👁️ 컴퓨터 비전의 목적
- 이미지 또는 영상 데이터에서 정보 추출
- 시각적 패턴 인식 및 분류
- 객체, 형태, 위치 등의 시각적 특징 분석

컴퓨터 비전은 **이미지를 단순히 저장·표시하는 것이 아니라**,  
데이터로 해석하고 분석하는 데 초점을 둡니다.

---

### 🧩 주요 처리 대상
- 정지 이미지(Image)
- 동영상(Video)
- 카메라 또는 센서로부터 수집된 시각 데이터

이러한 데이터는 픽셀 단위의 수치 정보로 구성되어 있으며,  
컴퓨터 비전은 이 수치들로부터 의미를 도출합니다.

---

### 🔍 대표적인 컴퓨터 비전 문제 유형
- 이미지 분류(Image Classification)
- 객체 탐지(Object Detection)
- 객체 분할(Image Segmentation)
- 특징 추출(Feature Extraction)
- 영상 처리(Video Analysis)

각 문제 유형은 목적과 출력 형태가 다르며,  
적절한 알고리즘과 처리 방식이 필요합니다.

---

### 🧠 컴퓨터 비전과 딥러닝
최근의 컴퓨터 비전 기술은  
**합성곱 신경망(CNN)을 포함한 딥러닝 모델**을 중심으로 발전하고 있습니다.

딥러닝 기반 접근 방식은  
사람이 직접 특징을 정의하지 않아도,  
데이터로부터 특징을 학습할 수 있다는 장점이 있습니다.

---

### ⚠️ 컴퓨터 비전의 한계
- 데이터 품질에 매우 민감함
- 조명, 각도, 해상도 변화에 영향받을 수 있음
- 학습 데이터에 없는 패턴에 대한 일반화 한계 존재

따라서 컴퓨터 비전 시스템은  
적절한 데이터 수집과 전처리 과정이 중요합니다.

---

## 컴퓨터 비전 실습

### 📌 실습 개요
컴퓨터 비전 실습에서는  
**이미지 데이터를 직접 처리하고 분석하는 기본 과정을 경험**하는 것을 목표로 합니다.

이 실습은 컴퓨터 비전의 원리를 이해하기 위한 기초 단계로 구성됩니다.

---

### 🧪 실습 주요 내용
- 이미지 불러오기
- 이미지 데이터 확인 및 시각화
- 픽셀 단위 데이터 구조 이해
- 간단한 이미지 처리 작업 수행

실습을 통해  
이미지가 숫자 배열로 표현된다는 점과  
처리 결과가 어떻게 변화하는지를 확인합니다.

---

### 🔄 기본 실습 흐름
1. 이미지 데이터 로드
2. 데이터 형태 및 크기 확인
3. 간단한 전처리 적용
4. 처리 결과 확인

해당 과정은  
이후 복잡한 컴퓨터 비전 모델을 이해하기 위한 기반이 됩니다.

---

### 🎯 실습의 목적
- 컴퓨터 비전 처리 흐름 이해
- 이미지 데이터 구조에 대한 감각 습득
- 이론과 실제의 연결

실습은 **정확한 성능 구현보다는 개념 이해**에 중점을 둡니다.

---

## 자연어 처리란? (Natural Language Processing, NLP)

### 📌 자연어 처리 개요
자연어 처리(NLP)는 **사람이 사용하는 언어(자연어)를 컴퓨터가 이해하고 처리할 수 있도록 하는 기술 분야**입니다.  
텍스트 또는 음성과 같은 언어 데이터를 분석하여,  
의미 있는 정보로 변환하거나 특정 작업을 수행하는 것을 목표로 합니다.

자연어 처리는 인공지능(AI)의 한 분야로,  
머신러닝과 딥러닝 기술의 발전과 함께 빠르게 발전해 왔습니다.

---

### 🧠 자연어 처리의 목적
- 텍스트 데이터의 의미 분석
- 언어 구조 및 문맥 이해
- 사람의 언어를 컴퓨터가 처리 가능한 형태로 변환

자연어 처리는 **문자를 단순히 나열된 데이터로 보지 않고**,  
의미와 관계를 가진 정보로 해석하는 데 중점을 둡니다.

---

### 🧩 자연어 처리의 주요 처리 대상
- 문서(Document)
- 문장(Sentence)
- 단어(Token)
- 음성 인식 결과 텍스트

이러한 데이터는 비정형 데이터에 해당하며,  
자연어 처리는 이를 구조화된 정보로 변환하는 역할을 합니다.

---

### 🔍 대표적인 자연어 처리 문제 유형
- 텍스트 분류(Text Classification)
- 감성 분석(Sentiment Analysis)
- 문서 요약(Text Summarization)
- 키워드 추출(Keyword Extraction)
- 질문 응답(Question Answering)
- 기계 번역(Machine Translation)

각 문제 유형은 입력과 출력 형태가 다르며,  
목적에 따라 서로 다른 처리 방식이 적용됩니다.

---

### 🔄 자연어 처리 기본 흐름
1. 텍스트 데이터 수집
2. 전처리(정제, 토큰화, 불용어 처리 등)
3. 텍스트 표현 변환
4. 모델 적용 또는 규칙 기반 처리
5. 결과 해석 및 활용

이 흐름은 대부분의 자연어 처리 작업에서 공통적으로 적용됩니다.

---

### 🧪 텍스트 전처리의 중요성
자연어 데이터는 잡음이 많기 때문에  
전처리 과정이 결과 품질에 큰 영향을 미칩니다.

- 불필요한 기호 제거
- 형태 단위 분리
- 중복 또는 의미 없는 단어 제거

전처리는 모델 성능 이전에 고려해야 할 핵심 단계입니다.

---

### 🧠 자연어 처리와 딥러닝
최근 자연어 처리 기술은  
딥러닝 기반 언어 모델을 중심으로 발전하고 있습니다.

딥러닝 기반 접근 방식은  
단어 간의 관계와 문맥을 학습할 수 있다는 장점이 있으나,  
데이터 품질과 양에 크게 의존합니다.

---

### ⚠️ 자연어 처리의 한계
- 문맥이나 뉘앙스 해석의 어려움
- 동일한 단어의 다의성 문제
- 학습 데이터에 포함되지 않은 표현에 대한 한계
- 결과의 완전한 정확성 보장 불가

따라서 자연어 처리 결과는  
상황에 따라 추가적인 검증이 필요합니다.

---

### 🎯 활용 목적
자연어 처리는 다음과 같은 목적을 위해 활용됩니다.

- 대량의 텍스트 데이터 분석
- 사용자 입력 이해 및 처리
- 정보 검색 및 요약 보조
- 언어 기반 서비스 기능 지원

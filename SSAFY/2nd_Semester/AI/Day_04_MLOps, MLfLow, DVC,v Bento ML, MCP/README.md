## MLOps

### 📌 MLOps 개요
MLOps(Machine Learning Operations)는  
**머신러닝 모델의 개발, 배포, 운영 전 과정을 효율적으로 관리하기 위한 방법론**입니다.

소프트웨어 개발에서의 DevOps 개념을 머신러닝 시스템에 확장한 것으로,  
모델이 실험 단계에 머무르지 않고 실제 서비스 환경에서 안정적으로 운영되도록 돕는 것을 목표로 합니다.

---

### 🎯 MLOps의 목적
- 모델 개발부터 배포까지의 흐름 정리
- 실험 결과 및 모델 버전 관리
- 재현 가능한 학습 환경 유지
- 운영 중인 모델의 변경 및 관리 용이성 확보

MLOps는 **모델 성능 자체보다 운영 안정성과 관리 효율성**에 초점을 둡니다.

---

### 🔄 MLOps의 일반적인 흐름
1. 데이터 수집 및 관리
2. 모델 학습 및 실험
3. 모델 평가
4. 배포 및 운영
5. 모델 변경 및 재학습

이 과정은 반복적으로 수행되며,  
각 단계가 체계적으로 연결되는 것이 중요합니다.

---

### ⚠️ MLOps의 필요성
- 모델 실험 결과가 많아질수록 관리 복잡도 증가
- 동일한 실험 재현의 어려움
- 개발 환경과 운영 환경 간 차이 문제

MLOps는 이러한 문제를 구조적으로 해결하기 위한 접근 방식입니다.

---

## MLflow

### 📌 MLflow 개요
MLflow는 **머신러닝 실험 과정을 기록하고 관리하기 위한 도구**입니다.  
모델 학습 과정에서 발생하는 다양한 실험 정보를 체계적으로 저장하고 추적하는 데 목적이 있습니다.

---

### 🧩 MLflow의 주요 기능
- 실험 파라미터 기록
- 학습 결과 및 지표 관리
- 모델 아티팩트 저장
- 실험 결과 비교

MLflow는 **실험 관리 도구**로서 활용되며,  
모델 성능을 자동으로 개선하는 역할을 하지는 않습니다.

---

### 🎯 활용 목적
- 여러 실험 결과의 비교 및 추적
- 모델 학습 과정의 기록 유지
- 실험 재현성 확보

---

## DVC (Data Version Control)

### 📌 DVC 개요
DVC는 **머신러닝 프로젝트에서 데이터와 모델 파일을 버전 관리하기 위한 도구**입니다.  
일반적인 소스 코드 버전 관리 도구로 다루기 어려운 대용량 데이터 관리에 초점을 둡니다.

---

### 🧩 DVC의 역할
- 데이터 파일 버전 관리
- 모델 결과물 관리
- 데이터 변경 이력 추적

DVC는 Git과 함께 사용되는 경우가 많지만,  
Git을 대체하는 도구는 아닙니다.

---

### 🎯 활용 목적
- 데이터 변경에 따른 실험 결과 추적
- 이전 실험 상태로의 복원
- 데이터 기반 실험의 재현성 확보

---

## BentoML

### 📌 BentoML 개요
BentoML은 **머신러닝 모델을 서비스 형태로 배포하기 위한 프레임워크**입니다.  
학습된 모델을 API 형태로 제공하는 과정을 단순화하는 데 목적이 있습니다.

---

### 🧩 BentoML의 역할
- 모델 패키징
- 추론 로직 관리
- 서비스 형태로 모델 제공

BentoML은 모델 학습 도구가 아닌,  
**배포 및 서빙을 지원하는 도구**입니다.

---

### ⚠️ 활용 시 주의점
- 모델 성능과 서비스 안정성은 별도의 고려 필요
- 배포 이후 모니터링 및 관리 전략 필요

---

## MCP

### 📌 MCP 개요
MCP는 **머신러닝 시스템에서 여러 구성 요소 간의 연계를 고려한 개념적 구조 또는 관리 접근 방식**을 의미합니다.  
모델, 데이터, 파이프라인 등 다양한 요소가 함께 작동하는 환경에서  
전체 흐름을 관리하는 관점으로 이해할 수 있습니다.

> MCP는 특정 도구나 표준을 의미하기보다는,  
> **시스템 전반을 바라보는 관리·설계 관점**으로 해석됩니다.

---

### 🎯 MCP의 관점
- 단일 모델이 아닌 전체 시스템 흐름 관리
- 데이터 → 모델 → 서비스 간 연결성 고려
- 변경 발생 시 영향 범위 파악

---

### ⚠️ 주의 사항
- MCP는 자동화 도구가 아님
- 구현 방식은 프로젝트 환경에 따라 달라질 수 있음
- 명확한 정의보다는 개념적 접근에 가까움
